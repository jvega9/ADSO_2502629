COMANDOS PRINCIPALES DE GIT

  • git init:Iniciar un proyecto en GIT.
  • git config --global user.email "email": identificar y configurar globalmente el email en un repositorio local.
  • git config --global user.name "nombre":identificar y configurar globalmente el nombre de usuario en un repositorio local.
  • git config --global user.name:Identificar el nombre de usuario en el repositorio local.
  • git config --global user.email:Verificar el email en el repositorio local.
  • git config --global --unset-all user.name Elimina las configuraciones del nombre de usuario en el repositorio local.
  • Clear: Borrar la pantalla del terminal.
  • cd nombre-directorio:Ingresa a un directorio en la terminal.
  • cd ..Salir del directorio en la terminal.
  • pwd:Mostrar la ruta del directorio actual en la terminal.
  • ls:Muestra la lista de contenidos de un directorio.
  • git log:Mostrar el historial de cambios en el repositorio.
  • git add .Agrega todos los archivos modificados al repositorio.
  • git status:Verificar el estado del repositorio.
  • git commit -m "comentario":Guaradar los cambios realizados en el repositorio con un mensaje resumido.
  • git branch:Mostrar el nombre de la rama.
  • git branch -m nombre_rama:Cambia el nombre de la rama.
  • git remote add origin url:Se agrega la direccion web del repositorio remoto
  • git push:Enviar los commits del repositorio local al repositorio remoto
